*WhatsSee* is a simple and humble image captioning application able to produce a text description of an image. \\
After extracting image features with pre-trained **VGG16** network, an **LSTM** network was built to generate captions. \\
I have used **Python3** and **Keras** for the back-end while the Web GUI front-end is built with **Flask** framework.
