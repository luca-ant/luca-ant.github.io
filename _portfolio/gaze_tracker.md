This project is an experiment on *gaze tracker* system. Based on the position of the **glint** on the iris, which is the reflection of a light source located near the camera, it is possible to estimate the point of the screen towards which you are looking. This information was then used as an **input system**, to move the mouse arrow. A **calibration phase** is needed before start controlling the mouse arrow.

*Gaze Tracker* is written in **Python3**. \\
Computer Vision algorithms provided by **OpenCV** library have been used for eye and pupil detection.
