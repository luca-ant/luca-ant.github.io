*Rubbish detection* is an application to classify images in order to separate waste for recycling. \\
The core has been developed in **Python3**, using **Tensorflow** for training. \\
After testing the performance of different **CNN** and fine-tuning them, the networks have been optimized with **TensorFlow Lite API** to run on Android.
